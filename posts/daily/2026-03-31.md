### 1. Anthropic Claude Code 源码泄露：超 51 万行代码意外公开

**事件**：Anthropic 在发布 Claude Code 2.1.8 版本时，因内部打包工具存在 BUG，导致超过 51.2 万行 TypeScript 源代码被意外打包进公开发布版本并公开泄露。源码包含模型行为指令、工具限制逻辑以及三层反蒸馏机制。Anthropic 随即展开法律反击，向 GitHub 发送 DMCA 下架通知，GitHub 一口气删除了 8100 个相关仓库——包括主仓库和大量 Fork。这是近年来 AI 行业规模最大的一次代码版权清理行动。
